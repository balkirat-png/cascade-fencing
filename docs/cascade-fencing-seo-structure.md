# Cascade Fencing — SEO Website Structure & Keyword Map

This document defines the recommended website structure, primary keywords, secondary keywords, and page-level targeting for a fencing company specializing in **wood fence installation** and **wood fence repair** in **Sumner, Washington** and surrounding cities.

---

# 1. Core Pages Overview

Your website should include the following primary pages:

1. Home
2. Wood Fence Installation
3. Wood Fence Repair
4. Specialty Fencing Styles
5. Service Area Hub
6. Gallery
7. Individual City Pages
8. About
9. Contact / Get a Quote

Each page is mapped with a primary keyword and several secondary keywords.

## 2.6 Gallery Page

The gallery is both a trust-building page for users and a supporting SEO asset.
It won't usually rank for core "hire a contractor" phrases, but it supports conversions and can catch photo-style searches.

**Primary Keyword**
- wood fence gallery

**Secondary Keywords**
- cedar fence gallery
- wood fence photos
- privacy fence photos
- horizontal wood fence pictures
- hog wire fence gallery
- board on board fence pictures
- fence installation before and after

**SEO Notes**
- Organize the gallery into sections:
  - Cedar privacy fences
  - Horizontal fences
  - Hog wire fences
  - Gates
  - Before-and-after projects
- Each section should have:
  - A short paragraph describing the style (about 100–150 words)
  - Links back to the Installation page and/or Specialty Styles page

**Internal Linking Recommendations**
- Link TO Gallery from:
  - Home page
  - Wood Fence Installation page
  - Specialty & Styles page
  - Service Area Hub page
- Link FROM Gallery to:
  - Wood Fence Installation page (main CTA)
  - Wood Fence Repair page (if you show repair before/afters)
  - Specialty & Styles page (for horizontal, hog wire, board-on-board)
  - Relevant City pages (e.g., "See more fences we've built in Bonney Lake and Lake Tapps")

**Conversion Strategy**
- Include a visible "Get Free Estimate" button at the top and bottom.
- Use lightbox/slideshow galleries to keep people on the page longer.
- Put the cleanest, best photos at the top of the page.

# 6. Internal Linking Strategy

- From the **Home page**, link to:
  - Wood Fence Installation
  - Wood Fence Repair
  - Service Area Hub
  - Gallery

- From the **Wood Fence Installation page**, link to:
  - Gallery
  - Specialty & Styles page
  - Top city pages (Bonney Lake, Puyallup, Lake Tapps, etc.)

- From the **Wood Fence Repair page**, link to:
  - Gallery (show repairs/before-after)
  - Service Area Hub

- From the **Gallery page**, link to:
  - Wood Fence Installation page (main CTA)
  - Wood Fence Repair page (where relevant)
  - Specialty & Styles page
  - Key city pages ("Fences we've built in Bonney Lake / Puyallup / Lake Tapps")

- From the **Service Area Hub**, link to:
  - Every city page

- From **City pages**, link back to:
  - Wood Fence Installation page
  - Wood Fence Repair page
  - Gallery page (if you show projects from that city)

This builds a clean internal structure and avoids keyword overlap.
